Sudoku – Rules

1. Objective of the Game

The goal of Sudoku is to fill a 9×9 grid so that every row, every column, and each of the nine 3×3 subgrids contain all digits from 1 to 9 exactly once.

2. Game Board

  • The board consists of 9 rows, 9 columns, and 9 blocks (each 3×3 cells).
  • Some numbers are pre-filled at the start.
  • The remaining cells must be filled using logical reasoning — guessing is not allowed!

3. Basic Rules

  • Each number may appear only once per row.
  • Each number may appear only once per column.
  • Each number may appear only once per 3×3 block.

Numbers must not be duplicated in any area.
